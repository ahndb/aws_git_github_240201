clear: 터미널 화면 지우기
pwd: 터미널의 현재 경로(위치)를 표시
ls: 현재 폴더의 파일 혹은 폴더 리스트를 보여줌
	-: 옵션 추가 명령어
	a: 모든 파일 및 폴더를 보여줌
	l: 파일 혹은 폴더의 상세 정보를 보여줌
ls -al 가능

- / rwx (1) / rwx (2) / rwx (3)
1 rwx : 소유주에 대한 권한
2 rwx : 소유그룹에 대한 권한
3 rwx : 기타 사용자(타인)
r : 읽기 /w : 작성 /x : 실행

cd: 폴더 이동
/ : 현재위치
. : 뒤(이전)

cd ../

mkdir 폴더명 : 폴더 생성 명령어
rm 파일명 : 파일||폴더를 삭제
r 옵션: 폴더 안에 하위 폴더와 파일까지 모두 삭제

vim 파일명(확장자포함) : vim editor를 사용하여 파일 생성
												vim editor를 사용하여 파일 열기
				vim editor 실행 직후는 읽기모드임
										-> 쓰기 모드로 변경하기 위해서는
										I or A 키를 입력하여 쓰기모드로 변경 가능
										-> 쓰기 모드에서 읽기 모드로 변경할 때는 ESC입력
										-> 읽기 모드에서 파일을 저장 및 종료하기 위해서는
insert - 입력
: 프롬포트 / w 저장 / q 종료 
/:wq 저장 후 종료 / q! 강제종료
